👋 Welcome to the Test Site

This is a dummy website for testing the PADME chatbot integration. The chatbot should appear in the bottom-right corner as a blue chat bubble.

What to test:

  • Click the chat bubble to open the chatbot
  • Try asking survey-related questions
  • Test boundary language responses
  • Check if definitions and explanations work
  • Test mobile responsiveness

🔧 Getting Started

Step 1: Get your Bot ID and Tenant ID from Copilot Studio

Step 2: Add credentials to the JavaScript section at the bottom of this file

Step 3: Refresh the page and click the chat bubble

Step 4: Test the chatbot interactions
